# Glossary

## Binary Mask

An image where each pixel is either background or plastic.

## Dashboard Contract

The agreed JSON/CSV field structure that the dashboard reads from the detection prototype.

## DINOv3

A modern vision foundation model family. In this project it appears through an EoMT segmentation model.

## DroidCam

An app that lets a phone camera act as a webcam or video stream.

## EoMT

Encoder-only Mask Transformer. A segmentation architecture that combines class queries and mask queries.

## IoU

Intersection over Union. A segmentation metric comparing predicted and ground truth overlap.

## mIoU

Mean Intersection over Union, averaged across classes.

## SegFormer

A transformer-based semantic segmentation model. It is the current default runtime model for the phone demo.

## Telemetry

Location and flight data such as latitude, longitude, altitude, and heading.

## Threshold

The cutoff used to convert plastic probability into a binary mask.

## Vibecoding

Using an AI coding agent as a collaborative coding partner while keeping changes guided by docs, constraints, and verification.
