# Release Checklist

Use this before a demo, tag, or handoff.

## Code

- Prototype starts.
- CLI help works.
- Output writer works.
- No accidental generated runs are staged.
- No model weights were changed unintentionally.

## Docs

- README is accurate.
- Getting Started is accurate.
- DroidCam tutorial is accurate.
- Dashboard contract is accurate.
- Troubleshooting includes recent known issues.

## Models

- SegFormer path exists.
- DINOv3 path exists.
- Model manifest is accurate.

## Demo

- Camera source tested.
- Save key tested.
- `latest_detection.json` tested.
- Manual location tested.

## Handoff

- Known limitations documented.
- Next steps documented.
- Changelog updated.
