# Local Skills

This folder contains project-local skill definitions. They are written as reusable workflows for Codex-style agents.

These skills are not a replacement for the global Codex skill system. They are local project playbooks that explain how to work consistently in this repository.

## Skills

- `droidcam-phone-demo`: Work on the phone/webcam inference prototype.
- `plastic-waste-dashboard`: Plan or build the dashboard around demo outputs.
- `model-inference`: Work on SegFormer and DINOv3 inference behavior.
- `project-docs`: Maintain the documentation system.

## How To Use

When a task matches a skill:

1. Read the matching `SKILL.md`.
2. Read the files it references.
3. Follow the verification steps.
4. Update docs if behavior changes.
